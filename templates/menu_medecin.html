Guide Utilisateur Medecin

En tant que médecin, vous avez accès à trois fonctionnalités principales dans notre application :

  1. Prediction multiple : Grâce à cette fonctionnalité, vous pouvez télécharger un fichier CSV contenant les variables nécessaires à la réalisation des prédictions pour tous vos patients en une seule fois. Pour utiliser cette fonctionnalité, suivez les étapes suivantes :
    • Cliquez sur "Prediction multiple" dans le menu.
    • Téléchargez votre fichier CSV contenant les informations de vos patients.
    • Notre système va alors générer les prédictions pour tous vos patients en une seule fois.
    • Les résultats seront disponibles dans la section "Historique".
  2. Prediction simple : En tant que médecin, vous pouvez également remplir la prédiction pour un patient spécifique en son nom. Voici comment faire :
    • Accédez à la section "Prediction simple" dans le menu.
    • Remplissez les informations requises pour le patient.
    • Après avoir soumis les informations, la prédiction sera générée et vous pourrez la consulter dans la section "Voir les prédictions".
  3. Historique : Cette fonctionnalité vous permet de voir une analyse des prédictions faites pour vos patients ou par vos patients. Elle vous donne une vue d'ensemble de la santé de vos patients basée sur leurs prédictions de santé.

Enfin, lorsque vous avez terminé votre session, vous pouvez vous déconnecter en cliquant sur "Se déconnecter" dans le menu.

Si vous avez des questions ou avez besoin d'aide, n'hésitez pas à nous contacter à tout moment.
